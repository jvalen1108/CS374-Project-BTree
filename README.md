# CS374-Project-BTree
B-Tree data structures Project with NBA Dataset

By Jose Valenzuela, Aiden Wells, and David Brickley

## Overview
An example of B-Tree implementation in Python, using NBA player statistics from 2004 to presesnt

## Files
- btree.py — B-Tree implementation with insert, search, delete, traverse, and range query
- driver.py — Demo driver that loads NBA data and tests all operations

## Dataset
NBA Database (1947–Present) by Eoin A. Moore, on Kaggle:
https://www.kaggle.com/datasets/eoinamoore/historical-nba-data-and-player-box-scores

## How to Run:
- Download PlayerStatistics.csv from the Kaggle link above
- Update the file path in driver.py to use your local copy
- Run python driver.py
